1. Project Overview

This project focuses on analyzing the Prime Video content library using Tableau to uncover insights related to content distribution, genre popularity, ratings, release trends, and geographical availability.
An interactive dashboard was developed to help stakeholders understand viewer-targeted content strategy and platform growth patterns.

The dashboard enables quick exploration of:

Content types (Movies vs TV Shows)

Audience ratings

Popular genres

Release year trends

Country-wise content distribution
